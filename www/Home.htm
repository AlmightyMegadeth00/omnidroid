OmniDroid is an event/action manager that allows users to automate system functionality based on incoming application intents. Applications will be able to plug-in configurations to allow our application (and therefore others) to interact with them.
                    "At last there is something that makes inter-application communication so easy".


 
 
 
 
OmniDroid is not just another Android application !!!!!
 
It is a framework for developers to work on and provides:
 
  - Flexibility between application communication
  - Automate event handling
  - Generic communication platform
