Refactoring the package name from edu.nyu.cs.omnidroid to edu.nyu.cs.omnidroid.app. This is a result of the original keystore used to sign the market app being lost or broken. So we had to rename the package instead. :-/

From now on I'll be the only one posting the app on the Market to avoid this happening again.
