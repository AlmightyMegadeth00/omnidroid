Refactoring the package name from edu.nyu.cs.omnidroid to edu.nyu.cs.omnidroid.app. This is a result of the original keystore used to sign the market app being lost or broken. Unfortunately the market doesn't have a way to handle the situation where the key is lost/corrupted. And although you can unpublish an app, you can't delete it or replace it. So we had to rename the package instead. :-/

From now on I'll be the only one posting the app on the Market to avoid this happening again.
